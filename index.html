O que é o HTML

HTML, Linguagem de Marcação de Hypertexto, é uma linguagem de marcação, usada para definir a estrutura do conteúdo de uma página web.

Exemplo de um HTML:

<p>Este é um parágrafo.</p>

Note o <p></p>, isto são tags, uma de abertura e a outra de fechamento, respectivamente.

Conhecendo algumas tags

Tags de Corpo do Documento

As tags <html>, <head>, e <body>, representam partes do documento.

A tag <html> representa todo o documento HTML, todos as demais tags são descendentes dela.

A tag <head> contém os metadados sobre o documento HTML, eles não são exibidos no navegador, porém carrega conetudos importantes para o funcionamento da página, como por exemplo os arquivos de CSS e JavaScript.

A tag <body> é o conteúdo legivel para o usuario no navegador, neles se encontra os textos, os links, entre outros.

Tags de Textos

Tags de títulos

Existem 6 elementos para títulos começando pelo <h1> e terminando no <h6>, no qual cada um representa um nível diferente de importância, onde, por exemplo o <h1> representa um título principal e o <h2> um subtítulo.

Tag parágrafo

A tag <p> é usada para representar um parágrafo.

Tag de links

A tag <a> é usada para criação de links.

Tag de links

<img>, usada para inserir imagens no HTML

Tag de Listas

Listas Ordenadas

As listas ordenadas, são os tipos de listas onde a ordem importa, ou seja, a posição do elemento tem um grau de importância, essas listas são númeradas, demonstrando assim a importancia de cada elemento na lista.

A tag <ol> é a responsavel por criar a lista ordenada.

Listas Não Ordenadas

As listas ordenadas, são os tipos de listas onde a ordem não importa, essas listas não são númeradas.

A tag <ul> é a responsavel por criar a lista não ordenada.
